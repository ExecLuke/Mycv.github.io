Fluid Design


This site is very simple and has very little information on each page. Using animate.css, I can create fluid animations for my elements. The simplistic design of this site makes it easy to add to and change when I want to.


Platform Responsive


This website will automatically scale and dynamically change elements to match your platform's visual aspects and keep a clean look. The nav bar at the top of each page will change into a drop down list, making it even simpler for the mobile user.


GitHub Pages


This site uses GitHub Pages and GitHub Desktop to automatically push updates to my repository in GitHub every time I make an update. GitHub Pages is a very simple way to manage and host your site. I can access and edit my sites files through any device and updates are instantly viewable on any device.
